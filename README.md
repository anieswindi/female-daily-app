# Project Title

Female Daily App

### Installation

1. Clone Repo

```bash
git clone -b master https://github.com/anieswindi/female-daily-app.git
```

2. Use the node package manager to install this.

```
npm install
```

3. React start scripts

```
npm start
```

### Usage
